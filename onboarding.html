Welcome to CF-Output-Line-Counter!

This extension displays line numbers on hover in Codeforces output fields, helping you track output results more easily.

Privacy Notice

This extension is designed with your privacy in mind. It does not collect, track, or store any of your personal information. It only accesses Codeforces pages to add a small visual enhancement on the output fields.

How to Use

1. Install the extension in your browser.
2. Go to any Codeforces problem page.
3. Hover over the output lines to see line numbers displayed on the left side.

Thank you for using CF-Output-Line-Counter! Happy coding!
